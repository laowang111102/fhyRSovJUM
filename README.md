# 图书管理系统 java460

## 项目概述

本项目是基于 SpringBoot + Vue 的图书管理系统，采用前后端分离的技术架构。系统旨在为用户提供便捷的图书借阅与管理系统，同时为图书管理员和管理员提供全面的功能模块。

## 技术栈

- 后端：SpringBoot、Shiro、MyBatis-Plus
- 前端：Vue、ElementUI、ECharts
- 数据库：MySQL
- 缓存：Redis
- 开发工具：IDEA、VSCode、Navicat

## 核心亮点

- 前后端分离架构，便于分工协作与维护
- 采用 SpringBoot + Vue 技术栈，开发效率高
- 集成 Shiro 框架，保障系统安全性与权限控制
- 使用 MyBatis-Plus，简化数据库操作
- 引入 Redis 缓存，提升系统性能
- 完善的功能模块，满足各类用户需求
- 易于扩展，方便后续功能升级与维护
